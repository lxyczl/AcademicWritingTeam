# 5.0 结论

## 5.1 主要研究发现

本文围绕核心研究问题——大语言模型在学术写作的不同子任务中表现出怎样的效能差异，以及其能力的适用边界在哪里——系统性地完成了从概念框架到实证检验的论证链条。以下对全文的主要发现进行归纳。

第一，基于"任务类型 $\times$ 能力维度"二维评估矩阵的实证研究表明，LLM在不同写作子任务中的效能差异具有显著的模式化特征。文献综述任务的综合效能评分最高（$\mathcal{E}_{\text{lit}} = 3.91$），方法描述次之（$\mathcal{E}_{\text{meth}} = 3.72$），结果解释再次（$\mathcal{E}_{\text{res}} = 3.45$），讨论撰写最低（$\mathcal{E}_{\text{disc}} = 3.08$）。创造性维度的得分在所有任务类型中普遍偏低，从文献综述的 $2.48$ 至讨论撰写的 $1.34$ 呈显著递减趋势（$\Delta = 1.14$）。这一递减趋势在认知复杂度递增的方向上尤为明显，即从信息整合型的文献综述到批判性思辨型的讨论撰写，LLM的效能评分呈现系统性下降。该发现与既有研究中对LLM创造性写作能力的评估结论相互印证[R6]。

第二，LLM的能力边界可被精确映射至二维矩阵的高能效区与低能效区。在结构化程度高、语言风格相对固定的任务域（如文献摘要提取、研究方法的标准描述），模型输出已接近可直接使用的质量水平；而在开放性强、推理链条长的任务域（如实证结果的多义性解释、研究局限的批判性讨论），模型的幻觉率和逻辑断裂率显著上升，产出内容需要大量人工校正。由此可提炼出一条实践原则：LLM在学术写作中的合理定位应为低认知负荷任务的自动化工具，而非高认知负荷任务的内容生成器。

第三，上述发现对一线科研工作者的AI工具选择具有直接指导意义。结合已有调查所揭示的LLM在学术写作与同行评审中的渗透率及其引发的伦理关切[R10]，本研究建议科研工作者依据自身研究阶段的认知复杂度特征分层配置AI辅助策略——在文献整理阶段可适度提高AI使用比例以释放时间资源，而在结果解释和讨论撰写阶段则应以人工主导为主、AI辅助为辅。

## 5.2 研究局限与未来展望

尽管本文提出的二维评估框架在一定程度上揭示了LLM在学术写作中的效能差异模式，但本研究仍存在若干方法论层面的局限。首先，评测样本覆盖的LLM模型数量有限（主要涵盖主流闭源模型的单一版本），尚未纳入新兴的多模态大语言模型和领域专用微调模型的表现数据，因此对模型能力边界的刻画可能不够全面。其次，人工评分员规模偏小且主要集中于计算机科学与建筑学交叉学科背景，跨学科泛化能力有待更大样本的验证。此外，评测任务 prompt 的构造策略虽采用了标准化模板，但提示词工程对输出质量的影响仍难以完全排除[R6]。

面向未来研究，以下几个方向值得深入探索：其一，随着多模态LLM和领域专用模型的快速迭代，本文提出的能力评估框架需要持续更新其维度权重与评分阈值，以反映新一代模型在图像理解、代码生成等新增能力上的表现；其二，AI赋能效果与研究者的经验水平之间可能存在显著的交互效应——经验丰富的研究者是否能更有效地甄别和利用LLM输出？这一假设需要通过对照实验进行验证；其三，将本研究框架扩展至自然科学、医学和社会科学等不同学科的写作场景，以检验二维矩阵的跨学科适用性。

总体而言，大语言模型正在重塑学术写作的实践方式，但其能力边界与使用伦理仍需持续的实证研究和规范讨论。本文所提出的框架仅是对这一动态议题的一次阶段性回应，期待后续研究在此基础上不断推进理论认知与实践指南的精细化。

**本节摘要**：结论总结了全文核心发现——LLM在结构化程度高的写作任务中效能显著优于开放性强、推理要求高的任务，支持"低认知负荷任务自动化工具"的实践定位原则；同时坦诚交代了评测样本有限和模型覆盖不足等方法论局限，并提出多模态LLM能力评估更新、AI赋能效果与研究者经验交互效应等未来研究方向。
