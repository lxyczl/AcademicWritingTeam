# AcademicWritingTeam
我构建了一套基于 Claude Code 的学术论文多 Agent 协同撰写系统，核心解决研究人员在论文写作中面临的三大痛点：文献真实性难保证、论证逻辑不连贯、引文格式与排版反复修改耗时。系统定义了五个专业化 Subagent（选题与文献调研、大纲与论证设计、分节内容撰写、体裁与格式规范、终审与质量审计），通过主协调 Agent 调度，形成从选题到终审的完整长链推理流程。
用户仅需输入研究兴趣和目标期刊，选题 Agent 便会发起联网搜索，批判性分析后生成附带真实文献的选题报告；大纲 Agent 基于报告绘制论证链条并输出结构化大纲；分节撰写 Agent 按大纲逐章写作，并通过上下文摘要维持全文逻辑连贯；格式 Agent 自动将引用统一为 GB/T 7714 顺序编码制，整理 LaTeX 公式并清除排版噪声；终审 Agent 则从学术诚信、逻辑结构和表达规范三个层面进行审计，发现问题时按失败层级自动退回修改，最多重试 2 次。各 Agent 间产物以 Markdown 文件传递，可检查、可回溯。目前该系统已成功完成《BIM 技术在建筑能耗预测中的应用》等投稿级初稿的自动生成，引文格式零返工，论证逻辑完整度显著提升，预计可将论文初稿撰写效率提高 80% 以上，并为后续团队协作提供了标准化的 AI 写作流水线。
